# Final Exercise Project - LEVEL 2 Machine Learning

## Project Name
Level 2 Integrated Machine Learning Challenge

## Objective
Build an end-to-end machine learning solution that demonstrates mastery of:
- Problem framing and metric selection
- Data preprocessing and feature engineering
- Model training, tuning, and evaluation
- Deployment-ready packaging and documentation

## Required Deliverables
1. Problem statement with business metric mapping
2. Reproducible training pipeline
3. At least 4 model candidates with comparison table
4. Hyperparameter tuning report
5. Error analysis and failure mode review
6. Basic prediction service interface (CLI or API)
7. Tests for critical pipeline components (minimum 20)
8. Clear README and architecture notes

## Evaluation Rubric
- Modeling correctness and rigor: 35%
- Evaluation quality: 25%
- Engineering quality: 20%
- Documentation and communication: 20%

## Graduation Gate for Level 2
A learner passes Level 2 when this final exercise project is complete, reproducible, and review-ready.
