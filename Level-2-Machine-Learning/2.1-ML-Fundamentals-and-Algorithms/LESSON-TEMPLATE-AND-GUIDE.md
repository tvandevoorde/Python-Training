# Lesson Template - LEVEL 2.1 ML Fundamentals and Algorithms

This template outlines the structure for LEVEL 2.1 ML Fundamentals and Algorithms lessons.

---

## Lesson Structure Overview

Each lesson follows this **8-hour format**:

| Section | Duration | Purpose |
|---------|----------|---------|
| Learning Objectives | 5 min | Set expectations |
| Concept and Workflow | 180 min | Build understanding |
| Practical Build | 120 min | Implement concepts |
| Exercises | 150 min | Practice and extension |
| Assessment + Homework | 25 min | Verify and reinforce |

**Total: ~8 hours per lesson**

Lesson-2.1.X-[Topic-Name].md
