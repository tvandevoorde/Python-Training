# Lesson 1.2.32: Gradients and Directional Derivatives

**Estimated Time**: 8 hours | **Difficulty**: Intermediate  
**Prerequisites**: Lesson 1.2.31 | **Topic**: Calculus for ML

---

## Learning Objectives

By the end of this lesson, you will be able to:
- Define the gradient vector
- Interpret steepest ascent
- Compute directional derivatives
- Connect gradients to optimization

---

## Core Ideas

- Gradient is a vector of partials
- Directional derivative projects gradient onto a direction

---

## Exercises

### Difficulty 1: Starter
- Exercise 1.2.32.1: Define gradient in one sentence.
- Exercise 1.2.32.2: Compute gradient of f(x,y)=x^2+y^2.
- Exercise 1.2.32.3: Explain steepest ascent.

### Difficulty 2: Intermediate
- Exercise 1.2.32.4: Compute directional derivative along a unit vector.
- Exercise 1.2.32.5: Explain why gradients point uphill.
- Exercise 1.2.32.6: Interpret gradient magnitude.

### Difficulty 3: Advanced
- Exercise 1.2.32.7: Compute gradient for a 3-variable function.
- Exercise 1.2.32.8: Explain gradient descent with gradients.

## Quiz & Assessment

### Quick Check (True/False)
1. Gradient is a vector of partials. (True)
2. Directional derivative uses unit direction. (True)
3. Gradient always points downhill. (False)
4. Zero gradient indicates a critical point. (True)

### Conceptual Questions
5. Why is gradient central to ML?
6. What does gradient magnitude tell you?
7. How is directional derivative computed?

### Coding Challenge
8. Compute gradient of f(x,y)=x^2+3y.

---

## Key Takeaways

- Gradient collects partial derivatives
- Directional derivative projects gradient
- Gradients guide optimization
- Zero gradient indicates stationary points
