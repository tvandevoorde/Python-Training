# Lesson 1.2.42: Loss Functions and Landscapes

**Estimated Time**: 8 hours | **Difficulty**: Intermediate  
**Prerequisites**: Lesson 1.2.41 | **Topic**: Calculus for ML

---

## Learning Objectives

By the end of this lesson, you will be able to:
- Identify common loss functions
- Explain convex vs non-convex losses
- Interpret loss landscapes
- Choose losses for tasks

---

## Core Ideas

- Loss defines the optimization target
- Shape of loss affects training difficulty

## Quiz & Assessment

### Quick Check (True/False)
1. Loss functions define training goals. (True)
2. All losses are convex. (False)
3. MSE penalizes large errors more. (True)
4. Loss landscape affects convergence. (True)

6. How does loss shape affect gradients?
7. What is a saddle point in a loss landscape?

## Key Takeaways

- Loss functions define objectives
- Convexity impacts optimization
- Loss choice affects learning behavior
