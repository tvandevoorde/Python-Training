# Lesson 1.2.30: Higher-Order Derivatives

**Estimated Time**: 8 hours | **Difficulty**: Intermediate  
**Prerequisites**: Lesson 1.2.29 | **Topic**: Calculus for ML

---

## Learning Objectives

By the end of this lesson, you will be able to:
- Compute second derivatives
- Interpret curvature and concavity
- Use second derivatives to find minima
- Relate to optimization

---

## Core Ideas

- Second derivative measures curvature
- Concave up vs concave down
- Second derivative test for minima/maxima

---

## Exercises

### Difficulty 1: Starter
- Exercise 1.2.30.1: Compute second derivative of x^3.
- Exercise 1.2.30.2: Explain concavity in words.
- Exercise 1.2.30.3: Identify where f''(x) > 0.

### Difficulty 2: Intermediate
- Exercise 1.2.30.4: Use second derivative test on a polynomial.
- Exercise 1.2.30.5: Explain inflection points.
- Exercise 1.2.30.6: Sketch a function and its second derivative.

### Difficulty 3: Advanced
- Exercise 1.2.30.7: Connect curvature to Hessian in ML.
- Exercise 1.2.30.8: Describe how curvature affects learning rate.
- Exercise 1.2.30.9: Explain convex vs non-convex loss.

## Quiz & Assessment

### Quick Check (True/False)
1. Second derivative measures curvature. (True)
2. If f''(x) < 0, function is concave up. (False)
3. Inflection points occur when curvature changes sign. (True)
4. Second derivative helps find minima. (True)

### Conceptual Questions
5. Why is curvature important for optimization?
6. How does concavity affect gradient descent?
7. What does f''(x)=0 indicate?

### Coding Challenge
8. Compute f''(x) for f(x)=x^3-3x.

---

## Key Takeaways

- Higher-order derivatives capture curvature
- Curvature guides optimization decisions
- Second derivative tests extrema
- Curvature relates to Hessian in ML
