# Final Exercise Project - LEVEL 1 Foundations

## Project Name
Level 1 Integrated Foundations Challenge

## Objective
Build one cohesive project that demonstrates mastery across LEVEL 1.1 to 1.4:
- Python programming fundamentals
- Mathematics reasoning for data tasks
- Data science tooling (NumPy, Pandas, visualization, Jupyter, SQLite)
- Project integration and delivery best practices

## Required Deliverables
1. Multi-module Python project structure
2. Data ingestion from at least one real source (API or public dataset)
3. Cleaning and transformation workflow with validation
4. SQLite data storage and at least 5 analytical SQL queries
5. 12+ high-quality visualizations with interpretation notes
6. One notebook and one script-based analysis flow
7. Tests for critical functions (minimum 15)
8. Logs and error handling for production-style robustness
9. README with setup, run steps, architecture, and findings
10. Reflection report (what worked, tradeoffs, next iteration)

## Evaluation Rubric
- Correctness and reliability: 30%
- Data quality and analytics depth: 25%
- Engineering quality (tests/logging/structure): 20%
- Communication quality (README/report): 15%
- Insight quality and recommendations: 10%

## Suggested Timeline (20 hours)
- Day 1: Scope and architecture
- Day 2: Ingestion + transformation + validation
- Day 3: Storage + SQL + visualization
- Day 4: Testing + documentation + polish

## Graduation Gate for Level 1
A learner passes Level 1 when this final exercise project is complete, reproducible, and review-ready.
