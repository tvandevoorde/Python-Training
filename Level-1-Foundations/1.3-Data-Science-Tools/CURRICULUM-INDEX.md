# LEVEL 1.3: Data Science Tools - Complete Lesson Index

**Total Time**: 600 hours | **Number of Lessons**: 75 lessons | **Lesson Duration**: ~8 hours each  
**Prerequisites**: LEVEL 1.2 (Mathematics Fundamentals) | **Target**: Practical data science tooling mastery

---

## Overview

This subLevel breaks down **Data Science Tools (600 hours)** into 75 eight-hour lessons, organized by topic.

**Learning Path:**
1. NumPy Mastery (19 lessons)
2. Pandas Data Manipulation (25 lessons)
3. Visualization with Matplotlib and Seaborn (13 lessons)
4. Jupyter Notebooks (12 lessons)
5. SQL Basics with SQLite (6 lessons)
