# LEVEL 1.1: Python Programming - Complete Lesson Index

**Total Time**: 700 hours | **Number of Lessons**: ~88 lessons | **Lesson Duration**: ~8 hours each  
**Prerequisites**: None | **Target**: Complete Python proficiency for data science

---

## Overview

This subLevel breaks down **Python Programming (700 hours)** into 88 eight-hour lessons, organized by topic.

**Learning Path:**
1. Variables, Types, Operators (10 lessons)
2. Control Flow (13 lessons)
3. Functions & Scope (15 lessons)
4. Data Structures (13 lessons)
5. File I/O, JSON, CSV (10 lessons)
6. Error Handling (13 lessons)
7. OOP Basics (13 lessons)
8. Modules & Packages (5 lessons)
9. Virtual Environments (5 lessons)
10. Testing Basics (5 lessons)

## Assessment Strategy

### Formative Assessment (ongoing)
- Lesson exercises (3 difficulty levels each)
- Mini-projects after each topic
- Quiz & assessment in each lesson

### Summative Assessment (milestone tests)
- Test after every 10-13 lessons
- 80+ points = Pass, continue
- <80 points = Review, retake

### Final Assessment (Capstone)
- Complete Python project on GitHub
- Demonstrates all 100 lessons
- Code review ready
- Portfolio piece
