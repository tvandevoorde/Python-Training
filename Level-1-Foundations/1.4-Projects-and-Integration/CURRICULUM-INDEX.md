# LEVEL 1.4: Projects and Integration - Complete Lesson Index

**Total Time**: 200 hours | **Number of Lessons**: 25 lessons | **Lesson Duration**: ~8 hours each  
**Prerequisites**: LEVEL 1.3 (Data Science Tools) | **Target**: Build and present integrated data projects

---

## Overview

This subLevel converts foundational skills into project delivery capability through two major projects and one final Level 1 exercise project.

**Learning Path:**
1. Project 1 Foundations and Build (Lessons 1.4.1 - 1.4.13)
2. Project 2 Portfolio Foundations and Build (Lessons 1.4.14 - 1.4.24)
3. Final Level 1 Exercise Project (Lesson 1.4.25)
