# Lesson Template - Projects and Integration

This template outlines the structure for all lessons in LEVEL 1.4 (Projects and Integration).

---

## Lesson Structure Overview

Each lesson follows this **8-hour format**:

| Section | Duration | Purpose |
|---------|----------|---------|
| Learning Objectives | 5 min | Set clear expectations |
| Conceptual Design | 120 min (2 hrs) | Architecture and decisions |
| Practical Build | 150 min (2.5 hrs) | Implementation walkthrough |
| Exercises | 150 min (2.5 hrs) | Guided and independent practice |
| Mini-Project/Project Work | 45 min | Integration and polish |
| Quiz & Homework | 30 min | Reinforcement |

**Total: ~8 hours per lesson**

---

## Design Principles

1. Build from reusable modules, not one-off scripts.
2. Prioritize validation, logging, and testability.
3. Communicate results with clear documentation.
4. Treat each lesson as a project delivery increment.
